Три основных блока в unit-тесте:

1. **Подготовка (Arrange):** В этом блоке осуществляется подготовка к выполнению теста. Это включает в себя создание всех необходимых объектов, инициализацию переменных, установку начальных условий и настройку окружения для теста.
    
2. **Действие (Act):** В этом блоке выполняется действие или операция, которую необходимо протестировать. Это может быть вызов метода или функции, который мы хотим протестировать, с передачей ему определенных аргументов.
    
3. **Проверка (Assert):** В этом блоке происходит проверка результата выполнения действия на соответствие ожидаемому результату. Здесь используются различные утверждения (assertions), чтобы убедиться, что возвращаемое значение или поведение тестируемого кода соответствует ожиданиям.
    

Пример:
```swift
func testAddition() {
    // Arrange
    let calculator = Calculator()
    let x = 3
    let y = 5
    
    // Act
    let result = calculator.add(x, y)
    
    // Assert
    XCTAssertEqual(result, 8)
}

```
- В блоке "Arrange" создается экземпляр калькулятора и задаются входные данные.
- В блоке "Act" вызывается метод `add` калькулятора с передачей ему аргументов `x` и `y`.
- В блоке "Assert" проверяется, что результат равен ожидаемому значению (8).
