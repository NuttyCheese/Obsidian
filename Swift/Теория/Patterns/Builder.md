Паттерн Builder (Строитель) в разработке программного обеспечения - это порождающий паттерн проектирования, который позволяет создавать объекты сложной структуры пошагово. Он позволяет создать объект с разными конфигурациями и опциями, не загромождая его конструктор, а также обеспечивает более гибкий и интуитивно понятный интерфейс для создания объекта.

Основные компоненты паттерна Builder:

1. **Builder (Строитель)**: Определяет общий интерфейс для создания различных частей сложного объекта.
   
2. **ConcreteBuilder (Конкретный строитель)**: Реализует интерфейс Builder и конкретно строит и возвращает объект. Также определяет методы для настройки объекта и его частей.
   
3. **Product (Продукт)**: Представляет объект, который строится. Обычно представляет собой сложную структуру, состоящую из нескольких частей.

4. **Director (Директор)**: Определяет порядок шагов для построения объекта, используя Builder.

В [[Swift]] паттерн Builder может быть реализован с использованием классов и протоколов. Это позволяет создавать объекты с различными конфигурациями, прозрачно для клиента, и обеспечивает удобство и гибкость при создании сложных объектов.
