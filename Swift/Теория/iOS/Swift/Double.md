**`Double`** — это **64-битный тип с плавающей точкой** (floating-point) в Swift, основной выбор для представления **дробных чисел** и **чисел с высокой точностью**.

Он хранит примерно **15–17 значащих цифр** и покрывает огромный диапазон значений: от ≈1.7×10⁻³⁰⁸ до ≈1.7×10³⁰⁸.

В 2026 году `Double` остаётся **де-факто стандартом** для большинства вычислений в iOS/macOS-приложениях.

### 1. Ключевые характеристики Double (актуально 2026)

| Характеристика                      | Значение / Особенность                                   | Сравнение с Float (32-бит)                         |
| ----------------------------------- | -------------------------------------------------------- | -------------------------------------------------- |
| Размер в памяти                     | 64 бита (8 байт)                                         | 32 бита (4 байта)                                  |
| Точность (значащих цифр)            | ~15–17                                                   | ~6–9                                               |
| Диапазон значений                   | ±1.7×10⁻³⁰⁸ … ±1.7×10³⁰⁸                                 | ±1.18×10⁻³⁸ … ±3.4×10³⁸                            |
| Минимальное положительное значение  | ≈2.225×10⁻³⁰⁸                                            | ≈1.4×10⁻⁴⁵                                         |
| Тип в [[Swift]]                     | [[struct]] ([[value type]])                              | То же                                              |
| Рекомендуемое использование         | Финансы, физика, графика, анимация, GPS, научные расчёты | Только в очень старом коде или для экономии памяти |
| Поддержка в [[SwiftUI]] / [[Metal]] | Полная (основной тип для анимаций, шейдеров)             | Почти не используется                              |

### 2. Самые частые способы создания Double

```swift
// 1. Литерал (самый частый)
let pi = 3.141592653589793          // Double по умолчанию
let gravity: Double = 9.81

// 2. Из других типов
let fromInt    = Double(42)          // 42.0
let fromFloat  = Double(Float(3.14)) // 3.14
let fromString = Double("123.456")   // Optional<Double> → 123.456

// 3. Экспоненциальная запись (scientific notation)
let avogadro   = 6.02214076e23       // ≈6.022×10²³
let verySmall  = 1.5e-45             // очень маленькое число

// 4. Специальные значения
let infinity   = Double.infinity     // ∞
let negInf     = -Double.infinity    // -∞
let nan        = Double.nan          // Not a Number
```

### 3. Основные операции и функции (топ-2026)

```swift
var a: Double = 10.0
var b: Double = 3.0

// Арифметика
let sum        = a + b          // 13.0
let diff       = a - b          // 7.0
let product    = a * b          // 30.0
let quotient   = a / b          // ≈3.3333333333333335
let remainder  = a.truncatingRemainder(dividingBy: b) // 1.0

// Округление
let rounded    = round(3.6)     // 4.0
let ceilVal    = ceil(3.1)      // 4.0
let floorVal   = floor(3.9)     // 3.0

// Математические функции (Foundation)
import Foundation

let sqrtVal    = sqrt(16.0)     // 4.0
let powVal     = pow(2.0, 10.0) // 1024.0
let sinVal     = sin(.pi / 2)   // 1.0
let cosVal     = cos(.pi)       // -1.0
let logVal     = log(1000.0)    // ≈6.907755 (натуральный логарифм)
```

### 4. Самые частые ловушки и как их избежать

| Ловушка                                       | Что происходит                                              | Как правильно в 2026 |
|-----------------------------------------------|-------------------------------------------------------------|----------------------|
| Сравнение на равенство                        | `0.1 + 0.2 == 0.3` → **false** (погрешность представления) | Используй `isApproximatelyEqual(to:)` или допуск ε |
| Деление на ноль                               | `10.0 / 0.0` → `inf` (бесконечность), не краш               | Проверяй `b != 0` перед делением |
| Накопление погрешности в циклах               | `sum += 0.1` 10 раз ≠ 1.0                                   | Используй `Decimal` для финансов |
| Парсинг строки → Double                       | `Double("3,14")` → `nil` (запятая вместо точки)             | Нормализуй строку: заменяй `,` на `.` |
| Сравнение с NaN                               | `nan == nan` → **false**                                    | Используй `.isNaN` |

### 5. Лучшие практики Double в Swift 2026

- **Для финансов** — **никогда** не используй `Double` → бери `Decimal`  
- **Для сравнения** — используй допуск (epsilon):

```swift
infix operator ≈ : ComparisonPrecedence
func ≈ (lhs: Double, rhs: Double) -> Bool {
    abs(lhs - rhs) < 1e-10
}
print(0.1 + 0.2 ≈ 0.3) // true
```

- **Для UI / анимаций** — `Double` идеален (Core Animation, SwiftUI использует его)  
- **Для научных вычислений** — `Double` — стандарт (Metal, Accelerate, simd)  
- **Swift 6 strict concurrency** — `Double` полностью [[Sendable]] и безопасен  
- **Документируйте** — пиши комментарий «Double — координата X в метрах (точность ≈15 цифр)»

**Короткий девиз 2026**:
> `Double` — это **основной тип** для дробных чисел в Swift: 64 бита, ≈15–17 цифр точности, огромный диапазон.  
> В 2026 году используй его для:  
> - графики, анимации, физики, GPS  
> - научных расчётов  
> - большинства математических операций  
> Но **никогда** — для денег (бери [[Decimal]]).
