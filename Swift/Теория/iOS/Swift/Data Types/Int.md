**`Int`** — это **основной целочисленный тип** в [[Swift]], который представляет **целые числа** (положительные, отрицательные и ноль).  
Это **[[value type]]** (структура), полностью безопасный и оптимизированный для большинства задач.

### 1. Ключевые характеристики Int (актуально 2025–2026)

| Характеристика                  | Значение / Особенность                                      | Примечание 2026 |
|---------------------------------|-------------------------------------------------------------|-----------------|
| Размер                              | 64-бит на 64-битных платформах (iPhone, Mac M-серии), 32-бит на старых 32-битных (редко) | `Int` = `Int64` на всех современных устройствах |
| Диапазон                            | -9 223 372 036 854 775 808 … +9 223 372 036 854 775 807     | `Int.min` … `Int.max` |
| Противоположность                   | `UInt` (беззнаковый, только ≥ 0)                            | `UInt` почти не используется |
| Value Type                          | Копируется при присваивании                                 | Нет скрытых ссылок |
| Signed                              | Да (может быть отрицательным)                               | Для беззнаковых — `UInt` |
| Overflow behavior                   | По умолчанию — **trap** (fatal error) в debug, **wrap** в release | Используй `&+`, `&-`, `&*` для wrap |

### 2. Создание и базовые операции

```swift
let positive: Int = 42
var negative = -100
let zero = 0

// Арифметика
let sum = positive + 10          // 52
let diff = positive - negative   // 142
let product = positive * 2       // 84
let quotient = negative / 5      // -20
let remainder = 42 % 10          // 2

// Сравнение
print(positive > negative)       // true
print(zero == 0)                 // true

// Инкремент/декремент
negative += 15                   // -85
positive -= 10                   // 32
```

### 3. Самые полезные идиомы Int в 2026 году

#### 3.1 Индексы массивов и циклы

```swift
let fruits = ["яблоко", "банан", "апельсин"]

for index in 0..<fruits.count {
    print("Фрукт \(index): \(fruits[index])")
}

// Более безопасно и читаемо
for (index, fruit) in fruits.enumerated() {
    print("[\(index)] → \(fruit)")
}
```

#### 3.2 Проверка чётности / нечётности

```swift
let number = 42
if number % 2 == 0 {
    print("Чётное")
} else {
    print("Нечётное")
}

// Более читаемо
let isEven = number.isMultiple(of: 2)
```

#### 3.3 Работа с диапазонами

```swift
for i in 1...5 {
    print(i) // 1 2 3 4 5
}

for i in stride(from: 0, through: 100, by: 10) {
    print(i) // 0, 10, 20, ..., 100
}
```

#### 3.4 Преобразования (самые частые)

```swift
// Int → String
let number = 123
let text = String(number)           // "123"

// String → Int (failable)
let input = "456"
if let value = Int(input) {
    print("Число:", value)          // 456
}

// Int → Double / Float
let doubleValue = Double(number)    // 123.0
let floatValue = Float(number)      // 123.0
```

### 4. Безопасная арифметика (overflow protection)

Swift **по умолчанию** ловит переполнение в debug-режиме:

```swift
let max = Int.max
// max + 1 // Fatal error в debug, wrap в release
```

Безопасные операторы (wrap around):

```swift
let a: Int = .max
let b = a &+ 1          // wrap → Int.min
let c = a &- 1          // wrap → Int.max - 1
let d = a &* 2          // wrap
```

**Рекомендация 2026**:  
В production используй обычные `+ - * /`, если переполнение невозможно (или проверяй диапазон).  
В низкоуровневом коде (битовые операции, крипто) — `&+ &*` и т.д.

### 5. Лучшие практики Int в Swift 2026

- **По умолчанию** используй `Int` (не `Int32`, `Int64` и т.д.) — это платформо-зависимый, но самый удобный тип  
- **Не используй** `UInt` почти никогда — отрицательные значения встречаются чаще, чем кажется  
- **Для индексов** — всегда `Int` (массивы, строки, циклы)  
- **Для идентификаторов** (ID в БД, [[UUID]]) — `Int64` или `String`  
- **Для возраста, количества** — `Int` + проверка `>= 0`  
- **Swift 6 strict concurrency** — `Int` полностью `Sendable` и безопасен  
- **Документируйте** — пиши комментарий «Int — количество попыток входа (≥ 0)»

**Короткий девиз 2026**:
> `Int` — это **рабочая лошадка** Swift: целые числа, индексы, счётчики, ID.  
> В 2026 году:  
> - используй `Int` по умолчанию (64-бит)  
> - избегай `UInt` и фиксированных размеров (`Int32`)  
> - проверяй диапазоны и валидность  
> - для математики и циклов — это **единственный правильный** выбор.
