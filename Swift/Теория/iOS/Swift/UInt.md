**`UInt`** — это **целое число без знака**, то есть может хранить только **неотрицательные числа** (0 и больше).

- Размер зависит от платформы:
    
    - **32-бит** на старых платформах
        
    - **64-бит** на современных платформах (например, на [[iOS]] устройствах)
        
- Используется там, где **отрицательные значения невозможны**, например:
    
    - индексы массивов
        
    - счётчики
        
    - размеры или длины
        

> Проще говоря: `UInt` = «целое число ≥ 0».

---

## 2. Основные термины

| Термин                            | Описание                                                                                                   |
| --------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| **Unsigned Integer**              | Целое число без знака (только положительные и ноль)                                                        |
| **[[Int]]**                       | Целое число со знаком (может быть отрицательным)                                                           |
| **Overflow**                      | Переполнение: при выходе за предел UInt, программа может аварийно завершиться без обработки (по умолчанию) |
| **UInt8, UInt16, UInt32, UInt64** | Разновидности UInt с фиксированным количеством бит                                                         |
| **init(_:)**                      | Инициализация из других чисел или типов                                                                    |
| **as**                            | Приведение типов между Int и UInt                                                                          |

---

## 3. Основной синтаксис

### Инициализация

```swift
let a: UInt = 10
let b = UInt(25) // через конструктор
```

### Сравнение с Int

```swift
let intNumber: Int = 5
let uintNumber: UInt = 5
// let sum = intNumber + uintNumber // Ошибка, нужно привести тип
let sum = UInt(intNumber) + uintNumber // правильно
```

---

## 4. Примеры от простого к сложному

### Пример 1. Простое использование

```swift
let count: UInt = 10
print("Count is \(count)")
```

- Просто создаём и выводим число
    

---

### Пример 2. Цикл с UInt

```swift
for i in 0..<10 as UInt {
    print(i)
}
```

- Можно использовать в циклах, но чаще используют `Int`
    

---

### Пример 3. Приведение типов

```swift
let intNumber: Int = 20
let uintNumber: UInt = UInt(intNumber)
```

- Нужно явное приведение между Int и UInt, чтобы избежать ошибок
    

---

### Пример 4. Использование с массивами

```swift
let array = ["a", "b", "c"]
let index: UInt = 1
print(array[Int(index)]) // UInt нельзя использовать напрямую, нужно привести
```

- UInt удобно хранить индексы, но для доступа к массиву нужен Int
    

---

### Пример 5. Безопасное сложение с проверкой переполнения

```swift
var x: UInt = UInt.max
// x = x + 1 // ошибка runtime
x = x &+ 1 // безопасное переполнение (wrap around)
print(x) // 0
```

- Обычное переполнение вызовет crash, `&+` делает безопасное «заворачивание»
    

---

## 5. Особенности UInt

1. **Только неотрицательные числа**
    
2. Размер зависит от платформы (32/64 бита)
    
3. Явное приведение нужно при взаимодействии с **Int**
    
4. Можно использовать фиксированные типы: `UInt8`, `UInt16`, `UInt32`, `UInt64`
    
5. Есть операции с безопасным переполнением: `&+`, `&-`, `&*`
    

---

## 6. Итог

- **UInt** = целое число без знака (≥ 0)
    
- Используется для:
    
    - Индексов
        
    - Счётчиков
        
    - Размеров и длин
        
- Требует **явного приведения при взаимодействии с Int**
    
- Поддерживает безопасное переполнение
