**`MKMapCameraZoomRange`** — это структура в **[[MapKit]]** (доступна с iOS 13.0+), которая задаёт **диапазон допустимого зума** (уровень приближения/удаления) для камеры карты `MKMapView`.

Она используется исключительно с **`MKMapCamera`** и позволяет ограничить, насколько близко или далеко пользователь может приблизить/отдалить карту.

Это полезно для:
- предотвращения слишком сильного зума (чтобы не показывать пикселизацию или пустоту),
- фиксации определённого уровня детализации (например, только городской масштаб),
- создания контролируемого UX (особенно в приложениях с навигацией, турами, AR).

### Основные свойства MKMapCameraZoomRange

```swift
public struct MKMapCameraZoomRange {
    public let minAltitude: CLLocationDistance?   // минимальная высота камеры (максимальное приближение)
    public let maxAltitude: CLLocationDistance?   // максимальная высота камеры (максимальное удаление)
}
```

- `minAltitude` — минимальная высота камеры над землёй (чем меньше — тем ближе зум)
- `maxAltitude` — максимальная высота (чем больше — тем дальше зум)
- Если значение `nil` — ограничение не применяется (безлимит)

### Типичные значения высоты (рекомендуемые диапазоны)

| Уровень зума              | minAltitude (метры) | maxAltitude (метры) | Примерный масштаб на экране          | Когда использовать |
|---------------------------|---------------------|---------------------|---------------------------------------|---------------------|
| Улица / здание            | 50–200              | 500–1000            | ~50–500 м                             | Детальный вид       |
| Городской / район         | 300–1000            | 3000–8000           | ~1–8 км                               | Городская навигация |
| Область / пригород        | 1000–5000           | 20 000–50 000       | ~10–50 км                             | Зона доставки       |
| Регион / страна           | 10 000–50 000       | 1 000 000–5 000 000 | ~100–5000 км                          | Обзорный вид        |
| Без ограничений           | `nil`               | `nil`               | От 10 м до всего мира                 | Полная свобода      |

### Как применять MKMapCameraZoomRange (самые популярные паттерны)

#### 1. Ограничение зума в определённом диапазоне (самый частый)

```swift
let camera = MKMapCamera()
camera.centerCoordinate = CLLocationCoordinate2D(latitude: 55.7558, longitude: 37.6173)
camera.pitch = 45
camera.heading = 0
camera.altitude = 1200

// Ограничиваем зум: от 200 м до 5000 м высоты
let zoomRange = MKMapCameraZoomRange(
    minAltitude: 200,     // нельзя приблизить ближе 200 м
    maxAltitude: 5000     // нельзя отдалить дальше 5 км
)

mapView.cameraZoomRange = zoomRange
mapView.setCamera(camera, animated: true)
```

#### 2. Только запрет слишком сильного зума (максимальное приближение)

```swift
// Запрещаем зум ближе 100 метров (например, чтобы не показывать пустые тайлы)
let zoomRange = MKMapCameraZoomRange(minAltitude: 100, maxAltitude: nil)

mapView.cameraZoomRange = zoomRange
```

#### 3. Только ограничение максимального удаления (обзорный вид)

```swift
// Не даём отдалять карту дальше 50 км (например, для приложения одного города)
let zoomRange = MKMapCameraZoomRange(minAltitude: nil, maxAltitude: 50_000)

mapView.cameraZoomRange = zoomRange
```

#### 4. Полное отключение ограничений (по умолчанию)

```swift
mapView.cameraZoomRange = nil  // или не задавать вовсе
```

### Лучшие практики MKMapCameraZoomRange в Swift 2026

- **Всегда** используйте **метры** (`minAltitude` / `maxAltitude`) — это интуитивнее и точнее, чем градусы  
- **Задавайте** `cameraZoomRange` **до** первого `setCamera` — изменения применяются мгновенно  
- **Для навигации** — часто ограничивают `minAltitude` ~100–300 м, чтобы не терять детали  
- **Для туристических приложений** — ограничивают `maxAltitude` до 50–100 км, чтобы не показывать весь мир  
- **Для AR / Flyover** — часто ставят `minAltitude` 50–100 м и `maxAltitude` 2000–5000 м  
- **В SwiftUI** — задавайте через `MapCameraPosition` или напрямую в `MKMapView` в `UIViewRepresentable`  
- **Privacy Manifest** (PrivacyInfo.xcprivacy) — обязательно с iOS 17+ для MapKit  
- **Документируйте** — пишите комментарий «MKMapCameraZoomRange — зум ограничен от 200 м до 5 км высоты камеры»

**Короткий итог 2026**:
> `MKMapCameraZoomRange` — это **диапазон допустимого зума** камеры карты (min/max высота в метрах).  
> В 2026 году:  
> - задаётся через `minAltitude` / `maxAltitude`  
> - применяется к `MKMapView.cameraZoomRange`  
> - самый удобный способ — метры (а не градусы)  
> - идеально для контроля детализации и предотвращения "пустых" тайлов  
> Это **единственный** способ ограничить уровень приближения/удаления в 3D-режиме карты.
