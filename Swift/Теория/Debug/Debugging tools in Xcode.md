Вот некоторые из основных инструментов отладки в Xcode:

1. **[[LLDB (Low-Level Debugger)]]:** Это стандартный отладчик, встроенный в [[Xcode]]. Он позволяет выполнять отладку кода на низком уровне, просматривать значения переменных, выполнять точные остановки и многое другое.

2. **[[Breakpoints (Точки останова)]]:** Позволяют остановить выполнение программы в определенной точке, чтобы проанализировать состояние приложения.

3. **[[Console (Консоль)]]:** Позволяет выводить текстовые сообщения из приложения для отладки и анализа.

4. **[[Variables View (Просмотр переменных)]]:** Позволяет просматривать значения переменных в текущем контексте выполнения программы.

5. **[[Debug Navigator (Отладочный навигатор)]]:** Позволяет просматривать стек вызовов, ресурсы памяти и другие данные о выполнении программы.

6. **[[Memory Graph Debugger (Отладчик графа памяти)]]:** Позволяет анализировать использование памяти в приложении и обнаруживать утечки памяти.

7. **[[Time Profiler (Профилировщик времени)]]:** Позволяет анализировать производительность приложения и выявлять узкие места в его выполнении.

8. **[[Network Debugging (Отладка сети)]]:** Позволяет анализировать сетевой трафик приложения для выявления проблем с сетью.

9. **[[Core Data Debugging (Отладка Core Data)]]:** Позволяет анализировать операции с базой данных Core Data для выявления проблем с сохранением данных.

Это только некоторые из инструментов отладки, доступных в Xcode. Каждый из них предназначен для решения определенных задач при отладке приложений.
