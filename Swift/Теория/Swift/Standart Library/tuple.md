## 1. Что такое `tuple`

**`tuple`** — это **группа значений, объединённых в один тип**.

- Может содержать **значения разных типов**
    
- Не создаёт отдельного типа — это просто структура для хранения нескольких элементов
    
- Часто используется для **возврата нескольких значений из функции**
    

> Проще говоря: tuple = «коробка с несколькими значениями».

---

## 2. Основные термины

| Термин                     | Описание                                                          |
| -------------------------- | ----------------------------------------------------------------- |
| **Element**                | Один элемент кортежа                                              |
| **Named tuple**            | Кортеж с именованными элементами (`(x: Int, y: Int)`)             |
| **Unnamed tuple**          | Кортеж без имён (([[Int]], [[String]]))                           |
| **Tuple decomposition**    | Распаковка кортежа в отдельные переменные                         |
| **Return multiple values** | Использование кортежа для возврата нескольких значений из функции |

---

## 3. Основной синтаксис

```swift
let point: (Int, Int) = (10, 20)
let namedPoint: (x: Int, y: Int) = (x: 10, y: 20)
```

- Можно использовать **именованные элементы** для удобного доступа
    
- Можно распаковывать кортежи в переменные
    

---

## 4. Примеры от простого к сложному

### Пример 1. Простейший tuple

```swift
let person = ("Alice", 30)
print(person.0) // Alice
print(person.1) // 30
```

- Доступ к элементам по индексу
    

---

### Пример 2. Named tuple

```swift
let point = (x: 10, y: 20)
print(point.x) // 10
print(point.y) // 20
```

- Удобнее обращаться по имени элемента
    

---

### Пример 3. Распаковка tuple

```swift
let person = ("Bob", 25)
let (name, age) = person
print(name) // Bob
print(age)  // 25
```

- Можно сразу присвоить элементы к переменным
    

---

### Пример 4. Возврат нескольких значений из функции

```swift
func minMax(array: [Int]) -> (min: Int, max: Int)? {
    guard let first = array.first else { return nil }
    var minValue = first
    var maxValue = first
    for value in array {
        if value < minValue { minValue = value }
        if value > maxValue { maxValue = value }
    }
    return (minValue, maxValue)
}

if let result = minMax(array: [3, 7, 1, 9]) {
    print("Min:", result.min, "Max:", result.max) // Min: 1 Max: 9
}
```

- Кортеж удобен для **возврата нескольких значений** без создания отдельной структуры
    

---

### Пример 5. Tuple в цикле и массиве

```swift
let points: [(x: Int, y: Int)] = [(1, 2), (3, 4), (5, 6)]
for (x, y) in points {
    print("x: \(x), y: \(y)")
}
```

- Можно хранить массив кортежей и распаковывать их в цикле
    

---

### Пример 6. Tuple с разными типами

```swift
let person: (name: String, age: Int, isMember: Bool) = ("Eve", 28, true)
print("\(person.name) is \(person.age) years old. Member: \(person.isMember)")
```

- Кортежи могут содержать **разные типы элементов**
    

---

## 5. Особенности tuple

1. **Value type** → копируется при присваивании
    
2. **Можно использовать разные типы элементов**
    
3. **Не создаёт отдельного типа**, но можно использовать **[[typealias]]**
    
4. Удобен для **возврата нескольких значений из функции**
    
5. Поддерживает **распаковку, именованные элементы и массивы кортежей**
    

---

## 6. Итог

- **Tuple** = группа значений, объединённых в один логический блок
    
- Поддерживает **индексы, имена, распаковку и хранение разных типов**
    
- Используется для **простого объединения нескольких значений или возврата из функции**
