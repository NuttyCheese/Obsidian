## 1. Что такое `let`

**`let`** — это ключевое слово для объявления **константы**.

- Константа — это **значение, которое нельзя изменить после инициализации**
    
- Используется для безопасного кода, где значение **не должно меняться**
    
- В отличие от [[var]], `let` гарантирует **immutability**
    

> Проще говоря: `let` = «переменная, которая после присвоения не меняется».

---

## 2. Основные термины

| Термин               | Описание                                                |
| -------------------- | ------------------------------------------------------- |
| **Immutable**        | Значение, которое нельзя изменить после присвоения      |
| **Type inference**   | Swift сам определяет тип константы, если не указан явно |
| **Explicit type**    | Указание типа константы (`let name: String = "Alice"`)  |
| **Shadowing**        | Переопределение константы внутри блока с тем же именем  |
| **Tuple binding**    | Использование `let` для распаковки кортежей             |
| **Pattern matching** | Использование `let` внутри [[switch]] и [[if case]]     |

---

## 3. Основной синтаксис

```swift
let name = "Alice" // Swift сам определяет тип String
let age: Int = 30  // Явное указание типа
```

- После присвоения `name = "Bob"` вызовет ошибку
    
- Тип можно указать явно или довериться компилятору
    

---

## 4. Примеры от простого к сложному

### Пример 1. Простейшая константа

```swift
let pi = 3.14159
// pi = 3.14 // ❌ Ошибка
print(pi)
```

- Значение `pi` нельзя изменить
    

---

### Пример 2. Кортеж с `let`

```swift
let point = (x: 10, y: 20)
print(point.x, point.y)
```

- Можно использовать для хранения нескольких значений
    

---

### Пример 3. Константа внутри массива

```swift
let numbers = [1, 2, 3, 4]
print(numbers[0]) // 1
// numbers.append(5) // ❌ Ошибка, так как массив объявлен через let
```

- **Immutable container** — нельзя изменять коллекцию, объявленную через `let`
    

---

### Пример 4. Shadowing

```swift
let value = 10
if true {
    let value = 20 // новая константа, shadowing
    print(value) // 20
}
print(value) // 10
```

- Внутри блока можно создать новую константу с тем же именем
    

---

### Пример 5. Pattern matching в switch

```swift
let number = 5
switch number {
case let x where x % 2 == 0:
    print("\(x) is even")
case let x:
    print("\(x) is odd")
}
```

- `let` позволяет захватывать значение внутри `switch`
    

---

### Пример 6. Распаковка [[Optional]]

```swift
let optionalName: String? = "Alice"
if let name = optionalName {
    print("Hello, \(name)")
}
```

- Безопасная распаковка Optional с использованием [[if let]]
    

---

## 5. Особенности `let`

1. **Immutable** — нельзя изменить после присвоения
    
2. **Тип выводится автоматически** или указывается явно
    
3. **Используется в pattern matching, optional binding и switch**
    
4. **Безопаснее [[var]]** — предотвращает непреднамеренные изменения
    
5. Shadowing позволяет временно переопределять внутри блока
    

---

## 6. Итог

- `let` = **константа, неизменяемое значение**
    
- Используется для **безопасного кода и оптимизации памяти**
    
- Можно использовать с кортежами, массивами, pattern matching и optional binding
    
- Основной инструмент для **[[immutable]] программирования** в Swift
