## 1. Что такое `var`

**`var`** используется для объявления переменной, значение которой можно **изменять** после инициализации.

- В отличие от [[let]], которое создаёт **константу**
    
- Может быть любого типа: [[Int]], [[String]], [[struct]], [[class]], [[Optional]]
    
- Работает с **локальными и глобальными переменными, свойствами классов и структур**
    

> Проще говоря: `var` = «переменная, которую можно изменять».

---

## 2. Основные термины

| Термин             | Описание                                                                |
| ------------------ | ----------------------------------------------------------------------- |
| **Mutable**        | Переменная может менять своё значение после инициализации               |
| **Type Inference** | [[Swift]] автоматически определяет тип по значению                      |
| **Optional**       | Переменная может быть Optional (`?`) и менять [[nil]]/значение          |
| **Scope**          | Переменная может быть локальной, глобальной, свойством класса/структуры |

---

## 3. Основной синтаксис

```swift
var age: Int = 25
age = 26
print(age) // 26
```

- `var` объявляет **изменяемую переменную**
    
- Тип можно **не указывать**, если значение ясно (`var age = 25`)
    

---

## 4. Примеры от простого к сложному

### Пример 1. Простая переменная

```swift
var name = "Alice"
name = "Bob"
print(name) // Bob
```

- Swift выводит тип `String` автоматически
    
- Значение можно менять
    

---

### Пример 2. Optional var

```swift
var nickname: String? = nil
nickname = "Ally"
print(nickname!) // Ally
```

- Optional можно изменять: nil → значение
    

---

### Пример 3. Массив как var

```swift
var numbers = [1, 2, 3]
numbers.append(4)
numbers[0] = 0
print(numbers) // [0, 2, 3, 4]
```

- Изменяемый массив через `var`
    
- Можно добавлять, удалять и менять элементы
    

---

### Пример 4. Свойство класса и структуры

```swift
struct Point {
    var x: Int
    var y: Int
}

var p = Point(x: 0, y: 0)
p.x = 10
print(p.x) // 10
```

- `var` внутри структуры или класса позволяет **изменять свойства**
    

---

### Пример 5. Сложный объект

```swift
class Person {
    var name: String
    var age: Int
    init(name: String, age: Int) {
        self.name = name
        self.age = age
    }
}

var person = Person(name: "Alice", age: 25)
person.age += 1
person.name = "Bob"
print("\(person.name), \(person.age)") // Bob, 26
```

- Изменяемые свойства через `var` внутри класса
    

---

## 5. Особенности var

1. **Mutable** — значение можно изменять
    
2. Может быть **Optional**
    
3. Используется в **локальной и глобальной области видимости**
    
4. В структурах `var` позволяет изменять свойства только если сам объект объявлен как `var`
    
5. Для классов `var` позволяет менять свойства независимо от того, mutable ли сам объект
    

---

## 6. Итог

- **var** = изменяемая переменная
    
- В отличие от `let`, **можно изменять значение после инициализации**
    
- Используется для:
    
    - Примитивных типов (`Int`, `String`)
        
    - Коллекций (`Array`, `Set`, `Dictionary`)
        
    - Свойств структур и классов
        
    - Optional и сложных объектов
        
- Позволяет писать **гибкий и изменяемый код**
