В [[Swift]] и [[UIKit]] "subviews" - это массив представлений (views), которые являются подвидами (subviews) определенного родительского представления (superview). 

Каждое представление ([[Swift/Теория/Swift/UIKit/UIView]]) в [[UIKit]] может содержать другие представления внутри себя. Представления, находящиеся внутри другого представления, называются его подвидами. Это может быть полезно, когда вам нужно объединить несколько элементов интерфейса в одно, создать сложные макеты или логически связанные компоненты.

Свойство `subviews` в `UIView` представляет собой массив всех представлений, которые содержатся в данном представлении. Вы можете добавлять и удалять представления из массива `subviews`, а также управлять их расположением и отображением.

Пример:

```swift
// Создаем родительское представление
let parentView = UIView(frame: CGRect(x: 0, y: 0, width: 200, height: 200))
parentView.backgroundColor = .lightGray

// Создаем подвиды
let subview1 = UIView(frame: CGRect(x: 20, y: 20, width: 100, height: 100))
subview1.backgroundColor = .red

let subview2 = UIView(frame: CGRect(x: 50, y: 50, width: 100, height: 100))
subview2.backgroundColor = .blue

// Добавляем подвиды в родительское представление
parentView.addSubview(subview1)
parentView.addSubview(subview2)

// Выводим все подвиды родительского представления
for subview in parentView.subviews {
    print(subview)
}
```

В этом примере создается родительское представление `parentView` с двумя подвидами `subview1` и `subview2`. Затем каждое из подвидов добавляется в родительское представление с помощью метода `addSubview(_:)`. Наконец, производится перебор всех подвидов `parentView` с помощью свойства `subviews`, и каждый из них выводится на консоль.
