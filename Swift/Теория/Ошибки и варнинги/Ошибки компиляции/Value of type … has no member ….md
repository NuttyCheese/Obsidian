#crash #fata_error #xcode #swift
### Что это значит

[[Xcode]] сообщает, что **вы пытаетесь обратиться к свойству, методу или элементу типа, которого не существует**.

- В [[Swift]] **типы строго типизированы**, и компилятор проверяет существование членов.
    
- Ошибка возникает при:
    
    1. Опечатке в имени метода или свойства.
        
    2. Попытке обратиться к члену, который не объявлен в типе.
        
    3. Использовании свойства/метода на [[Optional]] без извлечения значения.
        
    4. Попытке использовать член протокола или superclass, который не реализован в конкретном типе.
        

---

### Примеры кода/сценариев возникновения

**Пример 1: Опечатка в имени свойства**

```swift
struct Person {
    let name: String
}

let p = Person(name: "Alice")
print(p.nam) // ❌ Value of type 'Person' has no member 'nam'
```

- Опечатка `nam` вместо `name`.
    

---

**Пример 2: Метод не существует**

```swift
let numbers = [1, 2, 3]
numbers.add(4) // ❌ Value of type '[Int]' has no member 'add'
```

- Метод `add` отсутствует у `[Int]`. Нужно использовать `append`.
    

---

**Пример 3: Optional без извлечения**

```swift
let optionalPerson: Person? = Person(name: "Bob")
print(optionalPerson.name) // ❌ Value of type 'Person?' has no member 'name'
```

- `optionalPerson` — `Optional<Person>` → нужно использовать `?` или `!`:
    

```swift
print(optionalPerson?.name) // ✅ безопасно
print(optionalPerson!.name) // ✅ принудительное извлечение
```

---

**Пример 4: Член протокола/суперкласса отсутствует в конкретном типе**

```swift
protocol Greetable {
    func greet()
}

struct Dog: Greetable {
    // ❌ greet не реализован
}

let dog = Dog()
dog.greet() // ❌ Value of type 'Dog' has no member 'greet'
```

- Метод `greet` не реализован → компилятор не видит члена.
    

---

### Как исправить

#### 1️⃣ Проверить имя члена на опечатки

```swift
print(p.name) // ✅ правильно
```

---

#### 2️⃣ Использовать правильный метод/свойство для типа

```swift
numbers.append(4) // ✅ метод append существует для массивов
```

---

#### 3️⃣ Извлечь значение из Optional

```swift
print(optionalPerson?.name) // ✅ безопасно
```

---

#### 4️⃣ Реализовать требуемые методы/свойства протокола или суперкласса

```swift
struct Dog: Greetable {
    func greet() {
        print("Woof")
    }
}
```

---

### Резюме

- Ошибка возникает, когда **тип не содержит запрашиваемый член**.
    
- Исправляется через:
    
    1. Проверку имени свойства/метода.
        
    2. Использование корректного метода или свойства для типа.
        
    3. Извлечение значения из Optional.
        
    4. Реализацию методов/свойств протокола или суперкласса.
        
- Позволяет компилятору предотвращать доступ к несуществующим членам и избегать runtime crash.
