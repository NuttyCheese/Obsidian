#crash #fata_error #xcode #swift
### Что это значит

[[Xcode]] сообщает, что **вы пытаетесь использовать переменную до того, как она была инициализирована**.

- В [[Swift]] все переменные **должны иметь значение перед использованием**, если они не [[Optional]].
    
- Компилятор проверяет, чтобы переменные получали значение до первого обращения.
    
- Ошибка возникает при:
    
    1. Объявлении [[var]] без инициализации и последующем использовании.
        
    2. Логике с условными ветвями, где переменная может остаться неинициализированной.
        
    3. Использовании свойства класса/структуры без присвоения значения.
        

---

### Примеры кода/сценариев возникновения

**Пример 1: Простое использование без инициализации**

```swift
var x: Int
print(x) // ❌ Variable 'x' used before being initialized
```

- Переменная `x` объявлена, но не имеет значения.
    

---

**Пример 2: Условная инициализация**

```swift
var number: Int
let flag = false

if flag {
    number = 10
}

print(number) // ❌ Variable 'number' used before being initialized
```

- Если `flag == false`, `number` не получает значение → ошибка.
    

---

**Пример 3: Свойство класса без инициализации**

```swift
class Person {
    var name: String
}

let p = Person() // ❌ Variable 'name' used before being initialized
```

- Свойство `name` не имеет [[default]] значения и не инициализировано в [[init]].
    

---

### Как исправить

#### 1️⃣ Инициализация переменной при объявлении

```swift
var x: Int = 0
print(x) // ✅ теперь x инициализирован
```

---

#### 2️⃣ Гарантировать инициализацию во всех ветках кода

```swift
var number: Int
let flag = false

if flag {
    number = 10
} else {
    number = 0
}

print(number) // ✅ number гарантированно инициализирован
```

---

#### 3️⃣ Сделать переменную optional

```swift
var number: Int?
print(number) // ✅ безопасно, number = nil
```

---

#### 4️⃣ Инициализация свойств [[class]]/[[struct]] в init

```swift
class Person {
    var name: String
    
    init(name: String) {
        self.name = name
    }
}

let p = Person(name: "Alice") // ✅ name инициализирован
```

---

### Резюме

- Ошибка возникает, когда **переменная используется до присвоения значения**.
    
- Исправляется через:
    
    1. Инициализацию при объявлении.
        
    2. Гарантию инициализации во всех ветках кода.
        
    3. Использование optional, если значение может быть отсутствующим.
        
    4. Инициализацию всех свойств класса/структуры в init.
        
- Позволяет компилятору гарантировать корректное состояние переменных и избежать runtime crash.
