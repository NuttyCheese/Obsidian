`DispatchSemaphore` - это средство синхронизации, предоставляемое Grand Central Dispatch ([[GCD]]) в [[Swift]]. Он представляет собой объект счетчика, который позволяет управлять доступом к общему ресурсу из нескольких потоков. Семафор имеет внутренний счетчик, который уменьшается при вызове `wait()` и увеличивается при вызове `signal()`. Когда счетчик положителен, поток может продолжить выполнение; когда счетчик равен нулю, поток блокируется, ожидая, пока семафор не будет освобожден другим потоком.

Давайте рассмотрим подробнее как это работает с примерами:

1. **Создание семафора**:
   - Семафор можно создать с помощью конструктора `DispatchSemaphore`, указав начальное значение счетчика.

```swift
import Dispatch

let semaphore = DispatchSemaphore(value: 1) // Начальное значение счетчика равно 1
```

2. **Ожидание доступа к ресурсу**:
   - Для блокировки доступа к ресурсу поток вызывает метод `wait()` семафора. Если счетчик больше нуля, поток продолжает выполнение; в противном случае он блокируется.

```swift
semaphore.wait() // Блокирует поток, если счетчик равен 0
```

3. **Получение доступа к ресурсу**:
   - После успешного ожидания поток получает доступ к общему ресурсу и выполняет необходимые операции.

```swift
// Работа с общим ресурсом
```

4. **Освобождение ресурса**:
   - После завершения работы с ресурсом поток вызывает метод `signal()` для освобождения семафора и разрешения другим потокам получить доступ к ресурсу.

```swift
semaphore.signal() // Освобождает семафор и разрешает доступ другим потокам
```

Пример использования семафора для контроля доступа к общему ресурсу:

```swift
import Dispatch

let semaphore = DispatchSemaphore(value: 1) // Начальное значение счетчика равно 1

// Поток 1
DispatchQueue.global().async {
    semaphore.wait() // Блокирует поток, если счетчик равен 0
    // Работа с общим ресурсом
    semaphore.signal() // Освобождает семафор и разрешает доступ другим потокам
}

// Поток 2
DispatchQueue.global().async {
    semaphore.wait() // Блокирует поток, если счетчик равен 0
    // Работа с общим ресурсом
    semaphore.signal() // Освобождает семафор и разрешает доступ другим потокам
}
```

В этом примере `semaphore` используется для того, чтобы только один поток мог получить доступ к общему ресурсу одновременно. При этом остальные потоки будут ожидать, пока ресурс не освободится. Это обеспечивает безопасность работы с общими данными в многопоточной среде.
