#save_data #swift
`FileManager` в разработке на языке [[Swift]] представляет собой класс, который предоставляет интерфейс для взаимодействия с файловой системой устройства. С помощью `FileManager` вы можете выполнять операции чтения, записи, перемещения и удаления файлов и каталогов, а также получать информацию о файлах и каталогах.

Вот некоторые основные операции, которые можно выполнить с помощью `FileManager`:

1. **Чтение и запись файлов**: Вы можете использовать `FileManager` для чтения содержимого файлов или записи данных в файлы.
    
2. **Управление файлами и каталогами**: `FileManager` позволяет вам создавать, перемещать, переименовывать и удалять файлы и каталоги.
    
3. **Получение информации о файлах и каталогах**: Вы можете получить различную информацию о файлах и каталогах, такую как размер, тип, дата создания и другие атрибуты.
    
4. **Работа с ресурсами в приложении**: `FileManager` позволяет вам работать с ресурсами вашего приложения, например, копировать файлы из ресурсов приложения в файловую систему устройства.
    

Пример использования `FileManager` для чтения содержимого файла:
```swift
import Foundation

// Получаем путь к домашней директории
let homeDirectory = FileManager.default.homeDirectoryForCurrentUser

// Путь к файлу
let filePath = homeDirectory.appendingPathComponent("example.txt")

// Читаем содержимое файла
do {
    let fileContents = try String(contentsOf: filePath)
    print("Содержимое файла: \(fileContents)")
} catch {

}

```
В этом примере мы используем `FileManager.default`, чтобы получить экземпляр `FileManager`, и используем его для чтения содержимого файла по указанному пути. Если операция чтения завершается успешно, мы выводим содержимое файла на консоль. Если возникает ошибка, мы выводим сообщение об ошибке.
