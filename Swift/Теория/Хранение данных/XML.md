XML (Extensible Markup Language) - это универсальный язык разметки, предназначенный для структурирования данных и обмена информацией между различными системами. Он используется для описания иерархических структур данных с помощью тегов, атрибутов и текстовых значений.

Основные характеристики XML:

1. **Расширяемость**: XML позволяет определять собственные наборы тегов и структуры данных в соответствии с потребностями конкретного приложения или области применения.

2. **Читаемость для человека**: XML использует человекочитаемый формат, который упрощает визуальное восприятие структуры данных и ручное редактирование файлов.

3. **Поддержка Unicode**: XML поддерживает кодировку Unicode, что позволяет использовать широкий набор символов и языков при описании данных.

4. **Универсальность**: XML является универсальным форматом, который может использоваться во множестве областей, включая веб-разработку, обмен данными, конфигурацию приложений, настройки и многое другое.

XML часто используется в различных типах приложений, включая веб-сервисы, [[API]], базы данных, конфигурационные файлы и форматы обмена данными. В [[Swift]] для работы с XML можно использовать различные библиотеки и фреймворки, такие как [[Foundation]] или сторонние библиотеки, например `SWXMLHash`, которые предоставляют удобные средства для парсинга и создания XML-документов.
