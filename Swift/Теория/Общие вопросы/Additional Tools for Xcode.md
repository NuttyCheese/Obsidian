### [Дополнительные инструменты](https://developer.apple.com/download/all/) — Специализированный арсенал разработчика

##### ==Инструменты приватные, доступны только авторизованному пользователю==

Это не просто набор утилит, а целый арсенал для решения узких, но критически важных задач. Apple редко обновляет эти инструменты (часто они версионируются под конкретную ОС), но их ценность от этого не уменьшается. Скачать их можно в виде отдельного образа диска (`.dmg`), внутри которого находится несколько папок.

Вот подробный разбор того, что вы найдете внутри «Additional Tools for [[Xcode]]»:

#### 📁 Папка «Hardware IO Tools»

Это, пожалуй, самая полезная папка для тех, кто работает с реальными устройствами и симуляторами.

*   **Network Link Conditioner (для версий macOS до 10.13+)**:
    *   **Для чего:** **Критически важный инструмент для тестирования сетевой устойчивости.** После установки (через отдельный `.pkg` внутри папки) в настройках macOS (в разделе «Сеть») появляется панель для симуляции различных типов сетевого подключения.
    *   **Сценарий использования:** Вы можете включить профиль «3G» или «Edge» и проверить, как ваше приложение загружает данные, обрабатывает таймауты и ошибки соединения при плохом сигнале. *В современных версиях macOS (с 10.14+) эта функция встроена в симулятор iPhone или в настройки для разработчиков, но отдельный инсталлятор все еще доступен для легаси-систем.*
*   **Additional Tools for Xcode.dmg (внутри Hardware IO Tools)**:
    *   **Для чего:** Это "матрешка" с ещё более мелкими, но важными утилитами.
    *   **Что внутри:**
        *   **Accessibility Inspector:** Позволяет проверить, насколько ваше приложение доступно для людей с ограниченными возможностями (VoiceOver, динамический тип и т.д.). Показывает, какие элементы интерфейса некорректно размечены.
        *   **Audio QC:** Утилита для быстрой проверки аудиофайлов и кодеков.
        *   **Bluetooth Explorer:** Инструмент для анализа и отладки Bluetooth-соединений (полезно для разработки наушников или аксессуаров).
        *   **PacketLogger:** Мощнейший инструмент для захвата и анализа сетевого трафика на низком уровне. Позволяет видеть все пакеты, которыми обменивается ваше приложение (и система в целом), что незаменимо при отладке сложных сетевых протоколов.
*   **Запись экрана симулятора (simctl)**:
    *   **Для чего:** В этой папке часто лежат скрипты и утилиты, позволяющие записывать видео с экрана симулятора iPhone/iPad в формате, идеально подходящем для загрузки в App Store Connect (для видео-превью приложения). В новых версиях Xcode это делается одной кнопкой, но в старых — только так.
*   **Резинки для пультов (Crown Sequencer и др.)**:
    *   **Для чего:** Эмуляция вращения колесика Digital Crown на Apple Watch в симуляторе. Позволяет тестировать скролл и взаимодействие в приложениях для watchOS без физических часов.

#### 📁 Папка «Graphics Tools»

Здесь собраны утилиты для работы с графикой и [[Metal]] ([[API]] Apple для работы с графическим процессором).

*   **Metal Developer Tools**:
    *   **Для чего:** Набор для отладки и профилирования приложений, использующих Metal.
    *   **Внутри:**
        *   **Metal Capture:** Позволяет захватывать кадр за кадром то, что отрисовывает GPU, чтобы анализировать производительность и находить ошибки в шейдерах.
        *   **Texture Tool:** Конвертация и превью текстур в сжатые форматы, которые использует Apple (.ktx и другие).
*   **Quartz Composer** (устаревший, но все еще доступен):
    *   **Для чего:** Визуальная среда программирования для обработки графики в реальном времени. Раньше использовалась для создания заставок, визуализаций в iTunes и даже прототипов интерфейсов. Сейчас интересна скорее как артефакт истории или для поддержки старых проектов.
*   **OpenGL Profiler / Monitor**:
    *   **Для чего:** Инструменты для разработчиков, использующих устаревший OpenGL. Помогают увидеть, какие вызовы OpenGL выполняются и какова их стоимость для производительности.

#### 📁 Папка «Audio Tools»

Инструменты для профессионалов, работающих со звуком.

*   **AU Lab**:
    *   **Для чего:** Мощное приложение для тестирования и экспериментов с аудиокомпонентами (Audio Units — плагины, которые использует Logic Pro, GarageBand и др.). Позволяет создавать "микшерский пульт" из любых аудио-юнитов, тестировать задержки и маршрутизацию звука.
*   **Audio Unit Manager**:
    *   **Для чего:** Просмотр всех установленных в системе аудио-плагинов (Audio Units). Позволяет включать/отключать их, а также валидировать (проверять) свои собственные аудио-плагины, чтобы убедиться, что они корректно загружаются и работают без падений.

#### 📁 Другие полезные утилиты (могут лежать в корне образа)

*   **Accessibility Verifier (ACV)**:
    *   **Для чего:** Автоматизированная проверка приложения на соответствие стандартам доступности. Запускает ваше приложение и выдает отчет о проблемах (например, кнопка без описания для VoiceOver).
*   **CrashReporterPrefs**:
    *   **Для чего:** Маленькая утилита для включения расширенного логирования крашей (сбоев) приложения. Помогает получить больше информации о падении, если стандартных отчетов недостаточно.
*   **Console** (расширенная версия):
    *   **Для чего:** Хотя стандартная консоль есть в macOS, в Additional Tools иногда кладут бета-версии или версии с дополнительными фильтрами для разработчиков, позволяющие видеть системные логи в реальном времени с более тонкой настройкой.

> **Важно:** Многие из этих инструментов не обновлялись годами, но продолжают работать. Если вы разрабатываете под современные версии ОС, часть функционала (например, Network Link Conditioner) уже встроена в саму систему (включается в **Настройках -> Для разработчиков**). Однако для тонкой настройки или работы со старым оборудованием эти утилиты незаменимы.
