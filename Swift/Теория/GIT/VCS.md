### 1. Что такое VCS и зачем она нужна

Система контроля версий — это инструмент, который:

- хранит **историю всех изменений** файлов проекта  
- позволяет **откатиться** к любой предыдущей версии  
- даёт возможность **параллельной работы** нескольких разработчиков  
- помогает **отслеживать, кто, когда и что** изменил  
- упрощает **слияние** ([[merge and rebase#Вариант 1 — git merge (рекомендуется для общей ветки)|merge]]) и **разрешение конфликтов**  
- делает возможным **ветвление** (branching) для фич, экспериментов, хотфиксов

Без VCS разработка превращается в хаос: «кто сломал билд?», «где та версия, которая работала?», «как откатить?».

### 4. Почему Git победил всех (и продолжает побеждать в 2026)

- **Локальная история** — работа без интернета  
- **Дешёвое ветвление** — создание ветки = 1 файл в .git  
- **Скорость** — почти все операции локальные  
- **Экосистема** — GitHub, GitLab, Bitbucket, GitKraken, Sourcetree, Fork, Xcode Source Control  
- **Swift Package Manager** — работает только с Git  
- **Git LFS** — решил проблему бинарных файлов  
- **GitHub Copilot**, **GitLab Duo**, **Dependabot** — всё построено вокруг Git

### 6. Лучшие практики Git 2026 для iOS/Swift-команд

- **Ветки**: main (production), develop (интеграция), feature/…, bugfix/…, hotfix/…  
- **PR/MR**: размер 200–500 строк, шаблон, Conventional Commits  
- **CI/CD**: GitHub Actions / GitLab CI + swiftformat + swiftlint + тесты  
- **SPM**: приватные пакеты через GitHub Packages или GitLab Registry  
- **Зависимости**: Dependabot / Renovate + auto-merge patch/minor  
- **Хуки**: lefthook + pre-commit (swiftformat, swiftlint, тесты на изменённых файлах)  
- **.[[gitignore]]**: официальный [[Swift]].gitignore + Pods/ + DerivedData/ + .swiftpm/  
- **Релизы**: semantic-release или fastlane + автоматический changelog

**Короткий девиз 2026**:
> «Git + GitHub — это стандарт индустрии в 2026.  
> Если ты пишешь на Swift — ты почти наверняка уже используешь GitHub.  
> Всё остальное — либо legacy, либо специфические enterprise-кейсы.»
