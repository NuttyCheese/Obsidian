[[Xcode]] сообщает, что **компиляция [[Swift]]-исходников не завершилась успешно**.

- Это **общая ошибка компиляции**, которая появляется, когда Swift компилятор сталкивается с проблемой в коде или конфигурации проекта.
    
- Сообщение само по себе не даёт точной причины — нужно смотреть **предыдущие строки в логе сборки** для конкретной ошибки.
    
- Может возникать из-за:
    
    1. Синтаксических ошибок или ошибок типов.
        
    2. Проблем с зависимостями ([[CocoaPods]], [[SPM]], [[Carthage]]).
        
    3. Конфликтов модулей или импортов.
        
    4. Ограничений компилятора (например, слишком большой файл или сложный generic-код).
        
    5. Проблем с кэшами или Derived Data.
        

---

### Примеры

**Пример 1: Синтаксическая ошибка**

```swift
func greet(name: String {
    print("Hello, \(name)")
}
```

- Компилятор не сможет собрать файл → `Command CompileSwiftSources failed with a nonzero exit code`
    

**Пример 2: Несовпадение типов**

```swift
let x: Int = "10" // ❌ тип String не соответствует Int
```

- Ошибка типа в коде → компиляция не пройдена.
    

**Пример 3: Ошибки зависимостей**

- Использование устаревшего CocoaPod или несовместимой версии SwiftPM → компилятор не может собрать модуль.
    

---

### Как исправить

#### 1️⃣ Смотреть конкретную причину ошибки

- В Xcode: Report navigator → Build → открыть полный лог компиляции.
    
- Найти **первую строку с ошибкой** — именно она вызвала падение компиляции.
    

#### 2️⃣ Исправлять ошибки кода

- Синтаксис, типы, пропущенные [[return]], неверные вызовы функций.
    

#### 3️⃣ Очистка кэшей и Derived Data

```bash
rm -rf ~/Library/Developer/Xcode/DerivedData
```

- Иногда помогает при странных или старых зависимостях.
    

#### 4️⃣ Проверка зависимостей

- CocoaPods: `pod install` / `pod update`
    
- SwiftPM: `File → Packages → Reset Package Caches`
    
- Carthage: пересборка фреймворков.
    

#### 5️⃣ Разделение больших файлов (при необходимости)

- Иногда очень сложные [[generic]]-файлы вызывают падение компиляции.
    
- Разделение на несколько меньших файлов может помочь.
    

---

### Резюме

- Ошибка возникает, когда **компилятор [[Swift]] не может собрать исходники**.
    
- Исправляется через:
    
    1. Просмотр и исправление конкретных ошибок в логе компиляции.
        
    2. Очистку Derived Data и кэшей.
        
    3. Проверку и обновление зависимостей.
        
    4. Оптимизацию или исправление сложного кода.
        
- Позволяет успешно собрать проект и избежать непонятных падений сборки.
