[[Xcode]] сообщает, что вы **не передали обязательный аргумент функции или метода** при вызове.

- В [[Swift]] **каждый параметр функции, который не имеет default значения, обязателен**.
    
- Если аргумент пропущен → компилятор выдаёт ошибку.
    
- Часто возникает при:
    
    1. Вызове функции с несколькими параметрами, когда один или несколько аргументов пропущены.
        
    2. Неправильном использовании именованных параметров (label mismatch).
        
    3. Конструкторы классов/структур, где обязательный параметр не передан.
        

---

### Примеры кода/сценариев возникновения

**Пример 1: Функция с обязательным параметром**

```swift
func greet(name: String) {
    print("Hello, \(name)")
}

greet() // ❌ Missing argument for parameter 'name' in call
```

- Не передан аргумент `name`.
    

---

**Пример 2: Несовпадение именованных параметров**

```swift
func greet(name: String, age: Int) {
    print("\(name), \(age)")
}

greet(name: "Alice") // ❌ Missing argument for parameter 'age' in call
```

- Аргумент `age` обязателен, но не передан.
    

---

**Пример 3: Конструктор с обязательными параметрами**

```swift
struct Person {
    let name: String
    let age: Int
}

let person = Person(name: "Bob") // ❌ Missing argument for parameter 'age' in call
```

- Не передан обязательный параметр `age`.
    

---

### Как исправить

#### 1️⃣ Передать все обязательные аргументы

```swift
greet(name: "Alice") // ✅ корректно
greet(name: "Alice", age: 25) // ✅ корректно
let person = Person(name: "Bob", age: 30) // ✅ корректно
```

---

#### 2️⃣ Добавить default значения для параметров

```swift
func greet(name: String = "Guest", age: Int = 18) {
    print("\(name), \(age)")
}

greet() // ✅ теперь можно вызвать без аргументов
```

---

#### 3️⃣ Проверить правильность именованных параметров

```swift
greet(name: "Alice", age: 25) // ✅ имена параметров должны совпадать
```

---

### Резюме

- Ошибка возникает, когда **не передан обязательный аргумент функции, метода или конструктора**.
    
- Исправляется через:
    
    1. Передачу всех обязательных параметров.
        
    2. Добавление [[default]] значений параметрам.
        
    3. Проверку правильности именованных параметров.
        
- Помогает компилятору сопоставить вызов функции с её сигнатурой и избежать runtime ошибок.
