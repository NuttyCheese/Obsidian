**`continue`** — оператор управления потоком в [[Swift]], который **прерывает текущую итерацию цикла** и **переходит к следующей**.  
Используется внутри **циклов ([[for-in]], [[while]], [[repeat-while]])**, когда дальнейшее выполнение тела цикла не имеет смысла для текущей итерации.

---

## 🔹 Примеры кода

### 1. Пропуск четных чисел в цикле

```swift
for i in 1...5 {
    if i % 2 == 0 {
        continue // пропускаем четные числа
    }
    print("Нечетное число:", i)
}
```

---

### 2. Использование в `while`

```swift
var count = 0

while count < 5 {
    count += 1
    if count == 3 {
        continue // пропускаем выполнение тела цикла на 3
    }
    print("count =", count)
}
```

---

### 3. Пропуск пустых элементов массива

```swift
let items = ["apple", "", "banana", ""]

for item in items {
    if item.isEmpty {
        continue // пропускаем пустые строки
    }
    print("Элемент:", item)
}
```

---

### 4. Использование с метками (`label`) для вложенных циклов

```swift
outerLoop: for i in 1...3 {
    for j in 1...3 {
        if j == 2 {
            continue outerLoop // переходим к следующей итерации внешнего цикла
        }
        print("i = \(i), j = \(j)")
    }
}
```

---

### 5. Применение с условием и `guard`

```swift
let numbers = [1, 2, 3, -1, 5]

for num in numbers {
    guard num > 0 else {
        continue // пропускаем отрицательные числа
    }
    print("Обрабатываем число:", num)
}
```
