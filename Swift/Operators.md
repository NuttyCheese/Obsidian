**Operators** — это функции, которые изменяют, фильтруют или комбинируют потоки данных, созданные [[Publisher]]. Они не выполняют действие сразу, а создают новый Publisher, который работает с измененными данными. Чтобы "активировать" изменения, нужно подписаться на этот новый Publisher.

### Важные категории операторов в [[Combine]]

1. **Фильтрация и трансформация значений**
2. **Комбинирование нескольких Publisher**
3. **Обработка ошибок и завершений**
4. **Управление временем**

Давай по порядку разберем самые популярные операторы.

### 1. **Фильтрация и трансформация значений**

#### `map`

Оператор [[map]] позволяет преобразовывать каждое значение, которое издает Publisher. Он берет входное значение и преобразует его в другое.

Пример:

```swift
import Combine

let publisher = [1, 2, 3, 4].publisher

// Используем map для умножения каждого числа на 2
let subscription = publisher
    .map { $0 * 2 } // Преобразуем данные
    .sink { value in
        print("Получено значение: \(value)")
    }

// Вывод:
// Получено значение: 2
// Получено значение: 4
// Получено значение: 6
// Получено значение: 8
```

Здесь каждый элемент массива умножается на 2 с помощью оператора `map`.

#### `filter`

Оператор [[filter]] позволяет отфильтровать элементы, которые не удовлетворяют определенному условию.

Пример:

```swift
import Combine

let publisher = [1, 2, 3, 4, 5].publisher

// Используем filter для фильтрации четных чисел
let subscription = publisher
    .filter { $0 % 2 == 0 } // Только четные числа
    .sink { value in
        print("Получено значение: \(value)")
    }

// Вывод:
// Получено значение: 2
// Получено значение: 4
```

Здесь мы получаем только четные числа из массива.

#### `compactMap`

Если значения могут быть опциональными и вам нужно преобразовать их в определенный тип (и отфильтровать [[nil]]), то используется [[compactMap]].

Пример:

```swift
import Combine

let publisher = ["1", "2", "a", "3"].publisher

// Преобразуем строки в целые числа и фильтруем ошибки преобразования
let subscription = publisher
    .compactMap { Int($0) } // Преобразует строку в Int, если возможно
    .sink { value in
        print("Получено значение: \(value)")
    }

// Вывод:
// Получено значение: 1
// Получено значение: 2
// Получено значение: 3
```

Здесь строки, которые не могут быть преобразованы в числа (например, "a"), будут исключены из потока.

### 2. **Комбинирование нескольких Publisher**

#### `combineLatest`

Оператор [[combineLatest]] комбинирует значения из нескольких [[Publisher]], отправляя данные, когда оба Publisher издают новые значения.

Пример:

```swift
import Combine

let publisher1 = [1, 2, 3].publisher
let publisher2 = ["A", "B", "C"].publisher

// Объединяем значения из двух Publisher
let subscription = publisher1
    .combineLatest(publisher2) { num, letter in
        return "\(num)-\(letter)"
    }
    .sink { value in
        print("Получено значение: \(value)")
    }

// Вывод:
// Получено значение: 1-A
// Получено значение: 2-B
// Получено значение: 3-C
```

Здесь каждый раз, когда один из Publisher издает новое значение, комбинируются его значения с другим Publisher, и мы получаем строку, состоящую из числа и буквы.

#### `merge`

Оператор `merge` позволяет комбинировать несколько Publisher, и из каждого из них будут приниматься данные, как только они будут готовы.

Пример:

```swift
import Combine

let publisher1 = [1, 2, 3].publisher
let publisher2 = [4, 5, 6].publisher

// Комбинируем два Publisher, данные будут поступать по мере их готовности
let subscription = publisher1
    .merge(with: publisher2)
    .sink { value in
        print("Получено значение: \(value)")
    }

// Вывод:
// Получено значение: 1
// Получено значение: 2
// Получено значение: 3
// Получено значение: 4
// Получено значение: 5
// Получено значение: 6
```

Здесь данные из обоих Publisher объединяются в один поток, и значения поступают по мере их поступления.

### 3. **Обработка ошибок и завершений**

#### `catch`

Оператор [[catch]] позволяет перехватывать ошибки, которые могут возникнуть в потоке, и обрабатывать их.

Пример:

```swift
import Combine

let publisher = Fail<Int, Error>(error: NSError(domain: "Test", code: 1, userInfo: nil))

// Используем catch для обработки ошибок
let subscription = publisher
    .catch { error in
        Just(0) // Если ошибка, отправляем значение 0
    }
    .sink { value in
        print("Получено значение: \(value)")
    }

// Вывод:
// Получено значение: 0
```

Здесь ошибка, возникшая в Publisher, перехватывается, и вместо нее возвращается новое значение (`0`).

#### `tryMap`

Оператор [[tryMap]] позволяет выполнять операции с возможными ошибками. Если возникает ошибка, она будет проброшена дальше.

Пример:

```swift
import Combine

let publisher = ["1", "2", "3", "a"].publisher

// Преобразуем строки в числа с обработкой ошибок
let subscription = publisher
    .tryMap { str -> Int in
        if let num = Int(str) {
            return num
        } else {
            throw NSError(domain: "InvalidData", code: 1, userInfo: nil)
        }
    }
    .sink(
        receiveCompletion: { completion in
            switch completion {
            case .finished:
                print("Завершено успешно")
            case .failure(let error):
                print("Произошла ошибка: \(error)")
            }
        },
        receiveValue: { value in
            print("Получено значение: \(value)")
        }
    )

// Вывод:
// Получено значение: 1
// Получено значение: 2
// Получено значение: 3
// Произошла ошибка: Error Domain=InvalidData Code=1 "(null)"
```

Здесь строка "a" вызывает ошибку, которая перехватывается и выводится.

### 4. **Управление временем**

#### `debounce`

Оператор [[debounce]] используется, чтобы ограничить частоту получения значений из Publisher. Он "задерживает" вывод значений до тех пор, пока не пройдет указанное время без новых значений.

Пример:

```swift
import Combine

let publisher = PassthroughSubject<String, Never>()

// Применяем debounce
let subscription = publisher
    .debounce(for: .seconds(1), scheduler: DispatchQueue.main)
    .sink { value in
        print("Получено значение: \(value)")
    }

// Публикуем значения
publisher.send("a")
publisher.send("b")
publisher.send("c")

// Через 1 секунду получим только "c"
```

Здесь мы отправляем несколько значений, но только последнее ("c") будет выведено, потому что другие значения были слишком близки по времени.

### Заключение

**Operators** в Combine — это мощные инструменты для работы с потоками данных. Они позволяют:

- Трансформировать и фильтровать данные.
- Комбинировать несколько потоков.
- Обрабатывать ошибки и завершения.
- Управлять временем, например, с помощью `debounce`.

Combine дает много возможностей для работы с асинхронными потоками, и операторы позволяют легко и эффективно управлять этими потоками, делая код более чистым и читаемым.
