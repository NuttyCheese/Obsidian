**`Int`** — это **целое число** в [[Swift]].

- Может быть **положительным или отрицательным**
    
- Имеет **фиксированный размер**, зависящий от платформы (32-bit или 64-bit)
    
- Поддерживает **арифметические, сравнения и побитовые операции**
    
- Является **[[Value Type]]**, реализован через структуру ([[struct]])
    

> Проще говоря: `Int` = «целое число, которое можно хранить, изменять и использовать в вычислениях».

---

## 2. Основные термины

| Термин                          | Описание                                                           |
| ------------------------------- | ------------------------------------------------------------------ |
| **Value Type**                  | Int — это структура, копия значения при присвоении                 |
| **Signed Integer**              | Может быть отрицательным или положительным                         |
| **Unsigned Integer ([[UInt]])** | Только положительные числа, включая 0                              |
| **Overflow**                    | Переполнение при превышении максимального значения                 |
| **Type Conversion**             | Преобразование между Int и другими типами ([[Double]], [[String]]) |

---

## 3. Основной синтаксис

```swift
let a: Int = 10      // immutable
var b: Int = -5      // mutable

b += 15              // 10
print(a, b)          // 10 10
```

- `let` → значение нельзя изменить
    
- `var` → значение можно изменять
    

---

## 4. Примеры от простого к сложному

### Пример 1. Присвоение и арифметика

```swift
var x: Int = 10
var y: Int = 3

let sum = x + y      // 13
let diff = x - y     // 7
let product = x * y  // 30
let quotient = x / y // 3
let remainder = x % y // 1
```

- Стандартные арифметические операции
    

---

### Пример 2. Сравнение чисел

```swift
let a = 5
let b = 10

print(a < b)  // true
print(a == b) // false
print(a != b) // true
```

- Используем для **условных выражений и логики**
    

---

### Пример 3. Работа с UInt (беззнаковый Int)

```swift
let positive: UInt = 20
// let negative: UInt = -5 // Ошибка: отрицательное число невозможно

let sum: UInt = positive + 5
print(sum) // 25
```

- `UInt` хранит только **неотрицательные числа**
    

---

### Пример 4. Преобразование типов

```swift
let intNumber: Int = 10
let doubleNumber: Double = Double(intNumber)
let stringNumber: String = String(intNumber)

print(doubleNumber) // 10.0
print(stringNumber) // "10"
```

- Int легко преобразовать в **Double, [[Float]], String**
    

---

### Пример 5. Работа с диапазонами и циклом

```swift
for i in 1...5 {
    print(i)
}
// Output: 1 2 3 4 5

let numbers = Array(1..<5)
print(numbers) // [1, 2, 3, 4]
```

- Int используется для **цифровых диапазонов и индексов массивов**
    

---

### Пример 6. Побитовые операции

```swift
let a: Int = 5      // 0101
let b: Int = 3      // 0011

print(a & b) // 1 (AND)
print(a | b) // 7 (OR)
print(a ^ b) // 6 (XOR)
print(~a)    // -6 (NOT)
print(a << 1) // 10 (сдвиг влево)
print(a >> 1) // 2  (сдвиг вправо)
```

- Побитовые операции часто используются в **масках и низкоуровневых вычислениях**
    

---

## 5. Особенности Int

1. **Value type** — копируется при присвоении
    
2. **Signed integer** — может быть отрицательным или положительным
    
3. **Диапазон зависит от платформы** (`Int.min`, `Int.max`)
    
4. Можно использовать с **арифметикой, сравнениями, побитовыми операциями, циклами**
    
5. Swift безопасен — переполнение нужно обрабатывать через `&+`, `&-`, `&*`
    

---

## 6. Итог

- **Int** = целое число в Swift, value type
    
- Может быть **mutable ([[var]]) или [[immutable]] ([[let]])**
    
- Поддерживает арифметику, сравнения, побитовые операции
    
- Используется в **циклаx, индексах массивов, вычислениях и преобразованиях типов**
